## Easter Eggs

- **Konami Code (Home page):** Type `↑ ↑ ↓ ↓ ← → ← → B A` to enable retro color mode.
- **Logo 7-Click Secret (Home page):** Click the top-left logo 7 times quickly to reveal a hidden message.
- **Fake Terminal (Home page):** Press `~` (or `` ` ``) to open a fake terminal overlay.
	- Commands: `help`, `about`, `date`, `contact`, `clear`
- **404 Mini-Game (`/404.html`):** Catch the 404 button 3 times to unlock the hidden line.
