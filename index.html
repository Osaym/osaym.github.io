Projects.

As time has progressed I have worked on many different projects. Here is a quick list of them.

I decided to mess around with the now deprecated Discord.py Python module and create an (un)funny Discord bot which can be used to anger the people of Discord.   • 5thGradeUsernameGenerator: this is a project created by me during May of 2021. After the successful creation of EdgeLordBot, I decided to continue messing around with Discord.py and create a second Discord bot, 5thGradeUsernameGenerator. This website is written in HTML5 and CSS3, and hosted via Google Firebase. Any new HTML concepts I learn of or advancements I think of are added whenever.

  • OsaymPassgen: this is a project created by me during the summer of 2021. It is a customizable, CLI-based Python script allowing the user to create a strong password for use with accounts on any website. The goal was to create a generator similar to that of the one included in LastPass, however entirely written in Python.
